## node version 관리자인 n모듈 글로벌로 설치

```bash
sudo npm install -g n
```

## 노드 버전 설치: sudo n [버전명]

```bash
sudo n 12.16.1
```

## 노드버전 확인

```bash
sudo node --version
```
